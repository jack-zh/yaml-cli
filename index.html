    Link-pub

    link-pub是一个练习GO语言的实验性项目，目的是做一个提供给极客的网站，让大家在上面轻易的找到自己想要的资源，有趣的事情。 暂时用revel web项目开发。做平易的页面，做牛逼的内容，做方便的交流平台。https://github.com/jack-zh/link-pub

    link-pub 是一个公开的项目，托管在github上面，大家都可以参与，关于以后的发展，还是要看项目的成熟程度，现在只是一个练习的阶段 https://github.com/jack-zh/link-pub

    Installation

    To install:

    # 依赖包的安装，以及依赖问题
### 安装revel
    go get github.com/robfig/revel
#### 生成revel可执行文件
    go build -o $GOBIN/revel github.com/robfig/revel/cmd
### 安装mongodb的driver Mgo 先安装依赖bzr
    install bzr
    go get labix.org/v2/mgo
### 项目生成和运行
    revel new link-pub

    Quick Start

    启动项目，在8888端口:

    revel run link-pub
