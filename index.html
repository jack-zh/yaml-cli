    Link-pub

    yaml-cli是一个用yaml做配置命令行的命令行小工具，用yaml定义命令格式，用Python生成json逻辑，然后用C语言的解析json，利用readline接口做命令行，有比较准确的提示，错误提示等。还提供了一个web用的接口和简单的demo。https://github.com/jack-zh/yaml-cli

    Installation

    To install:

    make check   #对json生成文件进行检查，是否符合规范
make json    #根据yaml配置，生成json
make rpc     #根据配置，生成rpc接口函数
make out     #生成最终的二进制可执行文件
make clean   #清除.o文件盒可执行二进制文件
make clear   #清除所有的生成文件

    Information

    1：一个yaml文件代表一个模块，登陆一个模块用：login model——name

    2：命令的提示用TAB键，关键字用TAB可以提示，？（英文）代表完全下一步查看，是命令尾，代表可以回车结束

    3：quit退出当前模块
